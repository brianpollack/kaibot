# Welcome to Protovate KaiBot

KaiBot is your autonomous AI feature agent.
It watches for feature requests and implements them end-to-end -- planning, coding, and verifying -- so you can focus on what matters.

## How It Works

1. **Drop a feature file** into the `features/` folder or press New Feature on the web.
2. **KaiBot picks it up**, reads the description, and explores your codebase
3. **A plan is generated** with step-by-step checkboxes in the plan window
4. **Each step is executed** -- code is written, tests run, files committed
5. **The feature file is marked complete** with a summary of what changed

Happy building!
